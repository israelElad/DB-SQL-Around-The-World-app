
1. please enter my.conf file in the main directory, or create it if it doesn't exist. fill it with the following:
[connector_python]
host=localhost
user=<insert mysql user name>
password=<insert your mysql user password> 
port=3306

2. run main.exe, or main.py file using "python main.py" in cmd.

* trubleshooting:
if something doesn't work please try to install packages in requirements.txt using "pip install -r requirements.txt" from cmd after using "cd" command to get to the software's folder.
